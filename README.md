В программе доступно:

Графическое моделирование нейросети.

Слои:
- Создавать полносвязные слои.

Функции активации:
- ReLU.
- Сигмоидальная.
- Гиперболический тангенс.
- SoftMax.

Действия:
- Загрузить датасет (в формате csv с исключительно числовыми значениями).
- Инициализировать веса модели.
- Стандартизировать ранее загруженные данные.
- Обработать данные (режиим инференса).
- Сохранить модель.
- Загрузить модель.
- Ручной ввод.

Также доступен выбор устройства, на котором будет производиться работа нейросети.
Чтобы устройство было доступно, для него должен быть установлен драйвер OpenCL.

Управление:
Клик левой кнопкой мыши по слою = выделение слоя.</br>
Стрелка вправо/влево при выделенном слое = перемещения фокуса на соседний слой справа/слева.
Backspace/Del при выделенном слое = удаление слоя.
Backspace/Del + Shift при выделенном слое = удаление функции активации на этом слое.
Движение колесом мыши = движение по полю вперёд назад.
Enter/двойной клик по количеству нейронов = после этого действия вы можете изменить количество нейронов в выбранном слое с помощью клавиатуры.

Обучение модели:

Доступен график прогресса обучения модели.

Можно выбрать:
- Количество эпох для обучения.
- Скорость обучения.
- Размер батча.

Можно указать в процентах размер обучающей/валидационной выборки.

Можно выбрать функцию потерь:
- MSE (на конечном слое должна отсутствовать функция потерь SoftMax).
- MAE (на конечном слое должна отсутствовать функция потерь SoftMax).
- SoftMax (на конечном слое должна присутствовать функция потерь SoftMax).
- BCEWithLogits (на конечном слое должна присутствовать функция потерь сигмоидальная и должен быть 1 нейрон).

Можно выбрать диапазон отображения эпох.

Указывается количество уже пройденных эпох и минимальная на текущий момент ошибка.

Доступна возможность загрузки лучшей модели на данный момент (веса модели на эпоху с минимальной ошибкой, лучшая модель обновляется автоматически).

Доступна возможность протестировать модель на тестовом наборе данных и в результате получить лишь значение функции потерь на данных этого файла.

Доступна возможность начать/остановить обучение модели.

В качестве оптимизатора градиентного спуска используется Adam.
