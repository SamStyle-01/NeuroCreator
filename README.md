В программе доступно:

Графическое моделирование нейросети.

Слои:
- Создавать полносвязные слои.

Функции активации:
- ReLU.
- Сигмоидальная.
- Гиперболический тангенс.
- SoftMax.

Действия:
- Загрузить датасет (в формате csv с исключительно числовыми значениями).
- Инициализировать веса модели.
- Стандартизировать ранее загруженные данные.
- Обработать данные (режиим инференса).
- Сохранить модель.
- Загрузить модель.
- Ручной ввод.

Также доступен выбор устройства, на котором будет производиться работа нейросети.
Чтобы устройство было доступно, для него должен быть установлен драйвер OpenCL.

Управление:
Клик левой кнопкой мыши по слою = выделение слоя.</br>
Стрелка вправо/влево при выделенном слое = перемещения фокуса на соседний слой справа/слева.</br>
Backspace/Del при выделенном слое = удаление слоя.</br>
Backspace/Del + Shift при выделенном слое = удаление функции активации на этом слое.</br>
Движение колесом мыши = движение по полю вперёд назад.</br>
Enter/двойной клик по количеству нейронов = после этого действия вы можете изменить количество нейронов в выбранном слое с помощью клавиатуры.</br></br>

Обучение модели:</br></br>

Доступен график прогресса обучения модели.</br></br>

Можно выбрать:
- Количество эпох для обучения.
- Скорость обучения.
- Размер батча.
</br>
Можно указать в процентах размер обучающей/валидационной выборки.</br></br>

Можно выбрать функцию потерь:
- MSE (на конечном слое должна отсутствовать функция потерь SoftMax).
- MAE (на конечном слое должна отсутствовать функция потерь SoftMax).
- SoftMax (на конечном слое должна присутствовать функция потерь SoftMax).
- BCEWithLogits (на конечном слое должна присутствовать функция потерь сигмоидальная и должен быть 1 нейрон).
</br>
Можно выбрать диапазон отображения эпох.</br></br>

Указывается количество уже пройденных эпох и минимальная на текущий момент ошибка.</br></br>

Доступна возможность загрузки лучшей модели на данный момент (веса модели на эпоху с минимальной ошибкой, лучшая модель обновляется автоматически).</br></br>

Доступна возможность протестировать модель на тестовом наборе данных и в результате получить лишь значение функции потерь на данных этого файла.</br></br>

Доступна возможность начать/остановить обучение модели.</br></br>

В качестве оптимизатора градиентного спуска используется Adam.</br></br>
